When I say "add docstrings", please add docstrings in this format

```python
"""Description

Arguments
---------
arg1 : type1
    arg1 description

arg2 : type2
    arg2 description


Returns
-------
ret1 : ret_type1
    ret1 description
```

The "Returns" section can be omitted for methods that do not return
anything


When type hints are present make the types match the typehints to avoid
squiggly warnings
